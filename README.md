# Silent Fullscreen Capturer
This is a very simple app with a simple function. As the name suggests, it lets you take a full-screen screenshot without any annoying windows or tabs. You just press a hotkey, and boom — it's done.
But before I explain what features it has, let me explain why I'm building this.
## Origin of this app
The idea came from my daily use of the built‑in Windows screenshot tool. Imagine this: you're watching a live stream and want to take a screenshot. You press PrtScrn — but nothing pops up. So you think, "OK, whatever" and give up on taking the screenshot. And then, a few seconds later, the screenshot window suddenly appears. This usually happens when I finish a game and want to take a screenshot of my stats, but when the pop-up finally shows up, it's already too late — the screen I wanted to capture is gone. This drives me crazy, because it didn't just happen once or twice, but at least a dozen times. So I figured, why not write a small program to fix this once and for all? And also, the old Windows 10 machine cannot automatically save the picture after you take a screenshot. So every time I took one, it just... didn't. That's the reason that pushed me to create this program.
## Features
This app is relatively small, around 90MB. A unique feature is that the hotkey can be set by the user instead of a fixed one. No annoying pop-ups, no "Save As" dialogs, no "Are you sure?" windows. You press the hotkey, and it's done.<br>
You can change the save folder anytime. Browse to wherever you want — your desktop, a screenshot folder, or some deep directory you'll forget about a week later. It will remember.<br>
It lives in your system tray. Click the icon to open the settings window, or right-click to quit. You don't need to keep a window open all the time.<br>
The old Windows 10 machine? No problem. Screenshots are saved automatically to your chosen folder with timestamps. No manual saving. Just a folder full of screenshots.<br>
If something goes wrong — like you forgot to set the save path — it will quietly print an error in the console. No crash, no freeze, no pop-up. Just silence.<br>

# 静默全屏截图

这玩意儿特别简单，功能也就一个。顾名思义，就是让你全屏截图的时候，不会弹出那些烦人的窗口和选项卡。按下快捷键，"砰"——完事了。

但在介绍功能之前，先说说我为什么要做这个。

## 为啥要做这个

这想法来源于我平时用 Windows 自带截图工具时的体验。想象一下：你正看直播呢，想截个图。按下 PrtScrn——啥反应没有。你就想，"行吧，算了"，不截了。结果过了几秒，截图窗口突然弹出来了。

这种情况经常发生在我打完一把游戏，想截个战绩的时候。等弹窗终于出来了，黄花菜都凉了——我想截的画面早没了。真的给我整破防了，因为这不是一次两次，少说十几次了。所以我就想，干嘛不自己写个小软件，一劳永逸解决这破事？

而且那台老 Windows 10 机器，截完图不会自动保存。每次我截完，它就……没存上。这才是真正逼我写这个程序的理由。

## 功能特点

这软件不算大，90MB 左右。其中一个特色是快捷键可以自己改，不用死记硬背。没有弹窗烦你，没有"另存为"对话框，也没有"你确定吗？"这种废话。按下快捷键，完事。

保存文件夹随时能改。你想存桌面就存桌面，想存截图文件夹就存截图文件夹，或者存到某个一周后你自己都找不着的地方。它都给你记着。

它平时就待在系统托盘里。点一下图标打开设置窗口，右键就能退出。不用一直开个窗口占地方。

老 Windows 10 机器？没问题。截图自动存到你选的文件夹，文件名带时间戳。不用手动保存。就是一个塞满截图的文件夹，干净利落。

万一出点岔子——比如你忘了设保存路径——它会在控制台安安静静地报个错。不会崩溃，不会卡死，不会弹窗。总之，安静就完事了。

你要是遇到啥问题，尽管提 issue，我看到就会处理。

用着开心就行。
